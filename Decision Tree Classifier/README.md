## Overview
The model analyzes features of electronic waste items to classify them as **recoverable or non-recoverable**, supporting sustainable recycling and resource recovery.

## Tech Stack
- Python  
- Pandas, NumPy  
- Scikit-learn  
- Matplotlib, Seaborn  
- Jupyter Notebook  

## Workflow
Data preprocessing → Feature encoding → Train-test split → Decision Tree model training → Model evaluation (cross-validation, confusion matrix, classification report)
